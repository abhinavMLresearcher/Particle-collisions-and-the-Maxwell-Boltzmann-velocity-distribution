# Particle-collisions-and-the-Maxwell-Boltzmann-velocity-distribution
Spare time in realization of the maxwell boltzaman speed distribution curve through the simulation for molecules undergoing elastic collision in a two dimensional container.As time passes, the system reaches an equilibrium state after multiple elastic collisions leading to gaussian speed distribution of molecules, which validate the maxwell boltzaman speed distribution curve.<img width="872" height="522" alt="Maxwell distribution of the particle" src="https://github.com/user-attachments/assets/ee2adbb3-4aaf-4fee-be49-7626e97f5b44" />
